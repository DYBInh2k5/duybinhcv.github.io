## 💡 **Tips**

1. **Mobile Testing**: Always test pages on mobile devices

3. **Icon Library**: Using Font Awesome 6.4.0 - check their docs for more icons

## 🐛 **Troubleshooting**

- Menu not opening? Check browser console for JS errors
- Images not showing? Update image paths in `src/` folder
- Styles look weird? Clear browser cache (Ctrl+Shift+Del)
- Mobile layout broken? Check viewport meta tag
